Règlement WorldProno2014

Comment jouer

  1. S'inscrire et se connecter
  2. Pronostiquer la coupe du monde dans son entier dans l'onglet "Mon pronostic"
  3. Enregistrer vos pronos avant le 12 juin
  4. Devenez VIP en apportant vos 10€ pour valider vos pronos (attention tout pronostic non payé avant le 11 juin ne sera pas pris en compte dans le jeu)
  5. Suivre les résultats

Devenez un Very Important Player (VIP)

Etre VIP, c'est une opportunité de gagner la cagnotte spéciale coupe du monde 2014.
Il vous suffit d'apporter une participation de 10€ pour devenir VIP
Les 10% meilleurs pronostisqueurs se partageront la cagnotte.
A 50 joueurs, cela nous fait 5 gagnants à répartition croissante, soit 500€ de gains à se partager.

Une seule condition: connaitre Willy ou un de ses amis qui se portera garant de vous.

Attribution des points

Accumuler les points suivant les règles suivantes :
Pendant les phases de poules :
  • 3 points par pronostic correct (V1, N, V2)
  • 1 point bonus pour le score correct
Pour les équipes qualifiées :
  • 2 points par équipe qualifiée par phase
  • 2 points supplémentaires pour l'ordre pour les huitèmes
Pour la phase finale (des huitièmes à la finale)
  • si les 2 équipes sont les bonnes, même nombre de points que pour les poules soit +3/+4 points
  • Trouver le champion du monde (peu importe les matchs précédents) : 5 points.
Le total est sur 309 points possibles :
  • 192 points pour les phases de poules
  • 48 points pour les qualifications
  • 69 points pour la phase finale

  • Règlement: cette page
  • Mon pronostic : C'est ici que vous devez rentrer vos pronostics avant de les envoyer.
  • Les résulats : Classement des joueurs et points cumulés de la compétition
